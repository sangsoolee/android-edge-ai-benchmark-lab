# Results

This directory stores benchmark output from real device runs.

```
results/
├── raw/        ← CSV files pulled from device via adb
└── graphs/     ← Charts generated by scripts/analyze/plot_results.py
```

## How to pull from device

```bash
adb pull /sdcard/Android/data/com.edgeai.benchmark/files/results/ ./raw/
```

## How to generate charts

```bash
cd ../scripts/analyze
python plot_results.py --input ../../results/raw/ --output ../../results/graphs/
```
